## Style Reference

### Design Systems

**material3** — Material Design 3 (Google)
- Dynamic color: `--md-sys-color-primary`, `--md-sys-color-surface`
- Radius scale: 0/4/8/12/16/28px (None→ExtraSmall→Small→Medium→Large→Full)
- Elevation via tonal color overlay (not just shadow)
- Typography: Display/Headline/Title/Body/Label scale

**hig** — Apple HIG / Liquid Glass
- `backdrop-filter: blur(20px) saturate(180%)`
- `background: rgba(255,255,255,0.15–0.25)`
- `box-shadow: inset 0 1px 0 rgba(255,255,255,0.4), 0 8px 32px rgba(0,0,0,0.2)`
- `border: 1px solid rgba(255,255,255,0.2–0.4)`
- Radius: 16–24px; SF Pro / system-ui font

**fluent2** — Microsoft Fluent 2
- Acrylic: `backdrop-filter: blur(40px)`; Mica (tints from wallpaper)
- Token scale: `--colorNeutralBackground1` through `6`
- Radius: 4px (sm) / 8px (md) / 16px (lg)
- Segoe UI Variable font

**antd** — Ant Design
- Primary: `#1677FF`; success `#52C41A`; warning `#FAAD14`; error `#FF4D4F`
- Compact density — dense tables, small padding
- Radius: 6px default, `border-top: 2px solid #1677FF` on active tabs
- Geist / PingFang SC / Microsoft YaHei

**carbon** — IBM Carbon
- Monochrome: `#161616` bg, `#262626` surface, `#393939` layer
- `#0F62FE` blue accent
- IBM Plex Sans / Mono
- 2px borders, 0 border-radius on inputs, grid-always

### Visual Aesthetics

**neubrutalism**
- `border: 2–4px solid #000`
- `box-shadow: 4px 4px 0 #000` — hard offset, zero blur
- Fills: `#FFE500` / `#FF3F3F` / `#00FF87` (neon)
- `border-radius: 0–4px`; `font-weight: 700–900`

**glassmorphism**
- `background: rgba(255,255,255,0.15–0.2)`
- `backdrop-filter: blur(4–20px)`
- `border: 1px solid rgba(255,255,255,0.2)`
- `box-shadow: 0 8px 32px rgba(31,38,135,0.35)`
- Radius: 10–20px; vivid colorful backgrounds

**neumorphic**
- `background: #e0e5ec` (single mid-tone)
- `box-shadow: 6px 6px 12px #b8bec7, -6px -6px 12px #ffffff`
- No borders, no gradients
- Pressed: `box-shadow: inset 4px 4px 8px #b8bec7, inset -4px -4px 8px #fff`

**claymorphism**
- `background: #FF6B6B` (vibrant pastel)
- `border-radius: 24–40px`
- `box-shadow: inset -4px -8px 0 rgba(0,0,0,0.15), inset 4px 4px 0 rgba(255,255,255,0.4), 0 16px 40px rgba(0,0,0,0.2)`
- Inflated 3D rubber/clay look

**dark-minimal**
- `--bg-base: #0A0A0A`; `--bg-surface: #111111`; `--bg-elevated: #1A1A1A`
- `--border: rgba(255,255,255,0.08)`; `--accent: #6366F1`
- Radius: 8–12px; Inter/Geist; `letter-spacing: -0.01em`
- Linear/Vercel/Raycast aesthetic

**aurora**
- Dark base `#0A0A0A`
- Floating color orbs: `filter: blur(60–100px); opacity: 0.7`
- Colors: `#6366F1` / `#EC4899` / `#06B6D4` / `#F59E0B`
- `mix-blend-mode: screen` layering

**skeuomorphic**
- Realistic textures (CSS gradients simulating leather/metal/wood)
- Deep shadows: `box-shadow: 0 10px 30px rgba(0,0,0,0.5), inset 0 1px 0 rgba(255,255,255,0.3)`
- Glossy highlights: `background: linear-gradient(135deg, rgba(255,255,255,0.4) 0%, transparent 50%)`
- Embossed text: `text-shadow: 0 1px 0 rgba(255,255,255,0.3), 0 -1px 0 rgba(0,0,0,0.3)`
