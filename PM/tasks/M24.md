## Overview
Nowy styl "slate" — czysty, nowoczesny, Tailwind Slate palette. Obsługuje dark i light mode przez `data-color-scheme` attribute na `.screen`. Generyczny mechanizm — każdy przyszły styl może z niego skorzystać.

## Color Palettes
- **Dark:** bg `slate-900` (#0f172a), surface `slate-800` (#1e293b), text `slate-100`, accent `indigo-500` (#6366F1)
- **Light:** bg `slate-50` (#f8fafc), surface `white`, text `slate-900`, accent `indigo-600` (#4f46e5)

## Architecture Note
`color_scheme` to generyczny mechanizm: `html-builder.js` dodaje `data-color-scheme="dark|light"` na `.screen`. CSS reaguje selektorem `[data-color-scheme="dark"] .element.button { ... }`. Istniejące style ignorują atrybut (backward compat). `null` → styl używa własnego default.
