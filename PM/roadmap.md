# MockupMCP Roadmap

## Phase 1 — MVP
13 MCP tools, 10 element types, wireframe style, PNG export, stdio MCP, JSON storage, Docker container, preview server.

## Phase 2 — Rozbudowa
Full component library (30+), styles (material, ios, hand-drawn), NLP screen generation, templates, live preview with WebSocket, SVG/PDF export, auto-layout.

## Phase 3 — Zaawansowane
Code export (React/Flutter/SwiftUI), navigation flow, Mermaid export, SSE transport, grouping, layers, animated transitions, Docker Hub publish.
